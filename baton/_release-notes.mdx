<Update label="December 12, 2025">
### Connector configuration
- You'll now find an **Advanced settings** tab on each connector's setup page, where you can configure the attributes synced by the connector and manage whether the connector's syncs should be automatically paused if data anomalies are detected.

### Connector capability updates
- The [Docusign](/baton/docusign-v2) connector now syncs signing groups and supports provisioning for groups, signing groups, and permission profiles. The connector also now supports account deprovisioning.

</Update>
