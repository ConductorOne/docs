C1 provides identity governance for Helpjuice. Integrate your Helpjuice
instance with C1 for unified visibility and governance over user access.

## Capabilities

|Resource|Sync|Provision|
|-|-|-|
|Users|<Icon icon="square-check" iconType="solid" color="#c937ae"/>||
|Groups|<Icon icon="square-check" iconType="solid" color="#c937ae"/>||

The connector reads one Helpjuice account. It syncs that account's users and
groups, and the membership of each group, including smart groups. Every user
syncs as enabled: Helpjuice's API does not report an enabled or disabled
state on user records. The user's Helpjuice role (such as admin or viewer)
is carried on the user profile for visibility, but roles are not modeled as
separate access.

## Gather Helpjuice credentials

<Warning>
You need administrator access to your Helpjuice account to find its API key.
</Warning>

<Steps>
  <Step>
    Sign in to your Helpjuice account as an administrator.
  </Step>

  <Step>
    Open your account settings and find the **API key** for the account.
  </Step>

  <Step>
    Copy the API key, and note your account's API base URL. It is your
    Helpjuice address with the `/api/v3` path appended, for example
    `https://mycompany.helpjuice.com/api/v3`.
  </Step>
</Steps>

## Configure the Helpjuice connector

<Tabs>
  <Tab title="Cloud-hosted">
    Follow these instructions to use a built-in, no-code connector hosted by C1.

    <Steps>
      <Step>
        In C1, navigate to **Integrations** > **Connectors** and click **Add connector**.
      </Step>

      <Step>
        Search for **Helpjuice** and click **Add**.
      </Step>

      <Step>
        Choose how to set up the new Helpjuice connector.
      </Step>

      <Step>
        Set the owner for this connector.
      </Step>

      <Step>
        Click **Next**.
      </Step>

      <Step>
        Find the **Settings** area of the page and click **Edit**.
      </Step>

      <Step>
        Enter the Helpjuice credentials:

        - **Helpjuice API base URL**: your account's full API address,
          including the subdomain and the `/api/v3` path, with no trailing
          slash. For example, `https://mycompany.helpjuice.com/api/v3`.
        - **API key**: the API key from your Helpjuice account settings.
      </Step>

      <Step>
        Click **Save**.
      </Step>

      <Step>
        The connector's label changes to **Syncing**, followed by **Connected**. You can view the logs to ensure that information is syncing.
      </Step>
    </Steps>

    **Done.** Your Helpjuice connector is now pulling access data into C1.
  </Tab>

  <Tab title="Self-hosted">
    Follow these instructions to run the Helpjuice connector in your own
    environment.

    <Steps>
      <Step>
        Create secrets for the Helpjuice credentials.
      </Step>

      <Step>
        Configure the connector environment variables:

        - **BATON_BASE_URL**: your account's full API address, including the
          subdomain and the `/api/v3` path, with no trailing slash. For
          example, `https://mycompany.helpjuice.com/api/v3`.
        - **BATON_API_KEY**: the API key from your Helpjuice account
          settings.
      </Step>

      <Step>
        Deploy the connector using your standard self-hosted connector process.
      </Step>
    </Steps>

    **Done.** Your Helpjuice connector is now pulling access data into C1.
  </Tab>
</Tabs>
