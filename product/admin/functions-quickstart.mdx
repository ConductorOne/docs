## Overview

This quickstart guide helps you create and deploy your first function.

## Step 1: Create your first function

1. Go to **Platform** > **Functions**
2. Click **Create function**
3. Name it "Hello World"
4. Paste this code:

```javascript
export default async function handler(event, context) {
  return {
    statusCode: 200,
    body: JSON.stringify({ 
      message: 'Hello from ConductorOne!' 
    })
  };
}
```

5. Click **Save**

## Step 2: Test the function

1. Click **Test** in the function editor
2. Use the default test event
3. Click **Run test**
4. Verify the response shows your hello message

## Step 3: Deploy

1. Click **Deploy**
2. Select your environment
3. Click **Confirm**

Your function is now deployed and ready to use.
