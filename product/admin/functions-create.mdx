## Overview

Use custom functions to extend platform capabilities with your own business logic.

## Prerequisites

- Admin access to your tenant
- Basic understanding of JavaScript or Python

## Create a function

1. Navigate to **Platform** > **Functions**
2. Click **Create function**
3. Enter a name and description for your function
4. Select your runtime environment
5. Write your function code
6. Click **Save**

## Function structure

Functions follow a standard structure:

```javascript
export default async function handler(event, context) {
  // Your custom logic here
  return {
    statusCode: 200,
    body: JSON.stringify({ message: 'Success' })
  };
}
```

## Test your function

Before deploying, test your function:

1. Click **Test** in the function editor
2. Provide sample input data
3. Review the output and logs
4. Make adjustments as needed

## Deploy your function

Once tested, deploy your function:

1. Click **Deploy**
2. Select the deployment environment
3. Confirm the deployment

Your function is now live and ready to use.
